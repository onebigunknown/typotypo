# Privacy Policy for Typotypo

Effective date: July 20, 2026

Typotypo is a Figma plugin for formatting Russian and English interface text.

This policy explains what information the plugin accesses and how that information is handled.

## Information accessed by the plugin

Typotypo accesses text content, selection information, and properties of relevant Figma layers only when needed to perform the formatting requested by the user.

Text in a Figma document may contain personal or confidential information. Typotypo processes this content within the Figma plugin environment. The developer does not receive or retain the contents of Figma documents.

## Information collected by the developer

The developer of Typotypo does not collect personal data or document content through the plugin.

Typotypo does not:

- require an account or authentication;
- use analytics or tracking technologies;
- display advertising;
- create user profiles;
- sell or share personal data;
- send document text or layer information to external servers.

## Local settings

Typotypo may use Figma client storage to remember plugin preferences, including interface language, theme, selected text language, and enabled typography rules.

These preferences are stored through Figma for the operation of the plugin. They are not transmitted to or accessible by the developer.

Users can restore the plugin’s default settings from the Typotypo interface.

## Network access

Typotypo does not require network access to format text. Its manifest prevents the plugin from making network requests.

Document text, layer information, and plugin settings are not sent to external servers.

## External links

Typotypo includes an optional **Support the creator** link.

The link opens a third-party payment service only after the user chooses to follow it. The link does not include document text, layer information, or plugin preferences.

The third-party service operates under its own terms and privacy practices. Typotypo does not control how that service handles information provided directly by the user.

## Data retention

Because the developer does not receive document content, plugin settings, or personal data through Typotypo, the developer does not retain this information.

Preferences stored through Figma client storage remain subject to Figma’s platform behavior and the user’s Figma environment.

## Data security

Typotypo is designed to minimize data exposure by processing text within the Figma plugin environment and by operating without external network access.

Users should avoid including confidential information when submitting public bug reports or feature requests.

## Changes to this policy

This privacy policy may be updated if Typotypo’s functionality or data-handling practices change.

The effective date at the top of this document will be updated when material changes are made.

## Contact

For privacy questions, open an issue in the public repository:

https://github.com/onebigunknown/typotypo/issues

Do not include confidential document content, personal data, passwords, access tokens, or other sensitive information in a public issue.
